# STEP 解题法 — 电路分析标准解题流程

> 改编自 GCSE Physics 的 FIFA 法，专为电路分析设计。
> STEP = Survey（审题）→ Technique（定方法）→ Execute（执行）→ Prove（验证）

---

## 为什么需要标准解题流程？

电路分析题容易：
- ❌ 漏求初始值/稳态值
- ❌ τ求错（忘了求Rth）
- ❌ 公式记错符号
- ❌ 单位忘记带

**STEP法让你每一步都有检查点，避免这些错误！**

---

## STEP 法详解

### 🟦 S — Survey（审题，2分钟）

**目的**：理解题目，明确已知、未知、电路类型。

**标准动作**：

```
Step S1：读题三遍
- 第1遍：读懂电路结构（有几个电源？几个动态元件？）
- 第2遍：提取关键数据（数值+单位）
- 第3遍：明确待求量（求什么？uc？还是ic？）

Step S2：画电路简图（如果题目没给图）
- 标出所有元件值
- 标出开关动作时刻
- 标出待求量位置

Step S3：判断电路类型
- 是RC电路还是RL电路？
- 是求零输入、零状态还是全响应？
- 能否用三要素法？（直流激励一阶电路→可以！）
```

**输出**：
```
已知：[列出所有已知条件，带单位]
求：  [列出待求量]
类型：[RC/RL] + [零输入/零状态/全响应]
方法：[三要素法 / 零输入公式 / 零状态公式]
```

---

### 🟩 T — Technique（定方法，1分钟）

**目的**：选择最简便的解题方法，避免绕弯路。

**决策树**：

```
题目是直流激励一阶电路吗？
│
├── 是 → 优先用【三要素法】（万能！）
│   │
│   ├── 待求量是状态变量（uc/il）？
│   │   └── 是 → 直接套三要素公式
│   │
│   └── 待求量是非状态变量（vr/ic/vl）？
│       └── 是 → 先求uc/il，再用VCR求
│
└── 否（非直流激励 / 高阶电路）
    └── 用经典法（列方程求解）
```

**三要素法检查清单**：
```
✅ 是直流激励吗？（电压源/电流源是常量）
✅ 是一阶电路吗？（只有一个动态元件）
✅ 待求量写得出f(0+)、f(∞)、τ吗？
→ 如果✅，就用三要素法！
```

**常见方法对比**：

| 方法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 三要素法 | 直流一阶，任何量 | 快！直接套公式 | 只适用直流一阶 |
| 双零法 | 理解原理 | 物理意义清晰 | 慢，要算两次 |
| 经典法 | 任何情况 | 通用 | 慢，要解微分方程 |

---

### 🟨 E — Execute（执行，5-10分钟）

**目的**：按选定方法，分步计算，每步都有明确意义。

#### 三要素法执行模板

```
Step E1：求初始值 f(0+)
┌─────────────────────────────────────┐
│ 如果f是状态变量（uc/il）：          │
│   用换路定则：f(0+)=f(0-)        │
│   先求f(0-)：画0-电路（稳态）    │
│   → 电容开路，电感短路              │
│                                  │
│ 如果f是非状态变量（vr/ic/vl）：    │
│   画0+等效电路                      │
│   → 电容用电压源（uc(0+)）代替    │
│   → 电感用电流源（il(0+)）代替    │
│   用电阻电路方法求f(0+)            │
└─────────────────────────────────────┘

Step E2：求稳态值 f(∞)
┌─────────────────────────────────────┐
│ 画稳态等效电路：                    │
│   → 电容开路（uc(∞)有值，ic(∞)=0）│
│   → 电感短路（il(∞)有值，vl(∞)=0）│
│ 用电阻电路方法求f(∞)              │
└─────────────────────────────────────┘

Step E3：求时间常数 τ
┌─────────────────────────────────────┐
│ RC电路：                          │
│   1. 移走电容C                    │
│   2. 求剩余二端网络的Rth（戴维南等效电阻）│
│   3. τ = Rth·C                    │
│                                  │
│ RL电路：                          │
│   1. 移走电感L                    │
│   2. 求剩余二端网络的Rth           │
│   3. τ = L/Rth                    │
└─────────────────────────────────────┘
⚠️ 常见错误：直接用小R算τ！要用Rth！

Step E4：代入公式
┌─────────────────────────────────────┐
│ f(t) = f(∞) + [f(0+) - f(∞)]·e^(-t/τ)  │
│                                  │
│ 检查：                              │
│ - t=0时，f(0)=f(0+)？ ✓          │
│ - t→∞时，f(∞)=f(∞)？ ✓          │
└─────────────────────────────────────┘
```

#### 零输入响应执行模板

```
如果题目明确是零输入（无外加激励）：
uc(t) = uc(0+)·e^(-t/τ)        （RC）
il(t) = il(0+)·e^(-t/τ)        （RL）

然后：
- ic(t) = C·duc/dt = -(uc(0+)/R)·e^(-t/τ)
- vl(t) = L·dil/dt = -R·il(0+)·e^(-t/τ)
```

#### 零状态响应执行模板

```
如果题目明确是零状态（初始储能为零）：
uc(t) = uc(∞)·(1-e^(-t/τ))    （RC）
il(t) = il(∞)·(1-e^(-t/τ))    （RL）

然后：
- ic(t) = C·duc/dt = (uc(∞)/R)·e^(-t/τ)
- vl(t) = L·dil/dt = -R·il(∞)·e^(-t/τ)
```

---

### 🟪 P — Prove（验证，2分钟）

**目的**：检查答案是否合理，避免低级错误。

**标准验证动作**：

```
Step P1：量纲检查
┌─────────────────────────────────────┐
│ 电压 → 单位是V ✓                       │
│ 电流 → 单位是A ✓                       │
│ 时间常数 → 单位是s ✓                   │
│ 指数项e^(-t/τ) → 无量纲 ✓             │
└─────────────────────────────────────┘

Step P2：边界检查
┌─────────────────────────────────────┐
│ 检查t=0：                           │
│   f(0) = f(∞) + [f(0+)-f(∞)]·1   │
│        = f(∞) + f(0+) - f(∞)        │
│        = f(0+) ✓  ✓  ✓                │
│                                  │
│ 检查t→∞：                           │
│   e^(-∞) = 0                        │
│   f(∞) = f(∞) ✓  ✓  ✓              │
└─────────────────────────────────────┘

Step P3：物理意义检查
┌─────────────────────────────────────┐
│ 零输入响应：应该衰减到0 ✓              │
│ 零状态响应：应该从0上升到稳态值 ✓       │
│ 全响应：应该在初始值和稳态值之间 ✓      │
│ τ越大，变化越慢 ✓                     │
└─────────────────────────────────────┘

Step P4：特殊情况检查
┌─────────────────────────────────────┐
│ 如果f(0+)=f(∞)：                   │
│   → 暂态响应为0，没有过渡过程！        │
│   → f(t)=f(∞)（常数）✓               │
└─────────────────────────────────────┘
```

**验证不通过时的排查清单**：

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| t=0时不等于f(0+) | 公式记错 | 重新推导三要素公式 |
| 量纲不对 | τ求错 | 检查Rth计算 |
| 响应不衰减/不上升 | 符号错 | 检查f(0+)-f(∞)的符号 |

## 完整解题示例

### 例题：用STEP法求解

**题目**（例5-7风格）：
```
图5-19电路，t<0时已稳态，us=12V，R1=6Ω，R2=3Ω，C=0.5F。
t=0时开关闭合，求t>0时的uc(t)和ic(t)。
```

---

#### 🟦 S — Survey

```
已知：
- us = 12V（直流）
- R1 = 6Ω，R2 = 3Ω
- C = 0.5F
- t<0时稳态，t=0开关闭合

求：
- uc(t)（t>0）
- ic(t)（t>0）

类型：
- RC电路（有一个电容）
- 全响应（既有初始储能，又有外加激励）
- 直流激励 → 可以用三要素法！
```

---

#### 🟩 T — Technique

```
决策：
- 是直流激励一阶RC电路 ✓
- 待求量uc是状态变量 ✓
- 可以写出uc(0+)、uc(∞)、τ ✓

→ 选定方法：【三要素法】
```

---

#### 🟨 E — Execute

```
Step E1：求初始值uc(0+)
- t<0时稳态，电容开路
- uc(0-) = 分压：12V × [3/(6+3)] = 4V
- 换路定则：uc(0+) = uc(0-) = 4V ✓

Step E2：求稳态值uc(∞)
- t>0时，开关闭合，电路再达稳态
- 电容开路，等效电路：
  → us=12V正端 → R1 → R2 → us负端
  → 电容并联在R2两端
- uc(∞) = 12V × [3/(6+3)] = 4V
  （咦？uc(∞)=uc(0+)？）

Step E3：求时间常数τ
- 移走电容C
- 求Rth（从电容两端看进去的等效电阻）
  → R1和R2并联：Rth = (6×3)/(6+3) = 2Ω
- τ = Rth·C = 2 × 0.5 = 1s ✓

Step E4：代入公式
- uc(∞)=uc(0+)=4V
- 所以：uc(t) = 4 + [4-4]·e^(-t/1) = 4V
- 咦？电容电压不变？

等等，不对！
重新检查Step E2：
- 开关闭合后，电路被分为两个独立部分！
- 右边RC部分：R2和C串联，但R2左端被开关短路到电源正端
- 重新画电路...
```

**发现问题！** 重新审题，发现开关闭合后电路结构改变。

```
重新执行Step E2：
- 开关闭合后，电容C两端被直接连接到电源两端
- 所以uc(∞) = 12V（不是4V！）
- 重新计算：uc(∞) = 12V ✓

Step E4（修正）：
- uc(t) = 12 + [4-12]·e^(-t/1)
- uc(t) = 12 - 8·e^(-t)  V ✓
```

---

#### 🟪 P — Prove

```
Step P1：量纲检查
- uc(t)：12V - 8V·无量纲 = V ✓

Step P2：边界检查
- t=0：uc(0) = 12 - 8 = 4V = uc(0+) ✓
- t→∞：uc(∞) = 12V = uc(∞) ✓

Step P3：物理意义检查
- 开关闭合后，电容从4V充电到12V ✓
- τ=1s，1s后充电到63.2%×(12-4)+4 = 9.056V ✓

→ 答案合理！
```

**最终答案**：
```
uc(t) = 12 - 8e^(-t)  V  （t≥0）
ic(t) = C·duc/dt = 0.5 × 8e^(-t) = 4e^(-t)  A  （t≥0）
```

## 常见解题陷阱

### 陷阱1：τ求错

**错误**：直接用电路中的某个R算τ。
**正确**：一定要用与动态元件连接的戴维南等效电阻Rth！

### 陷阱2：忘记了0+等效电路

**错误**：认为所有量都满足换路定则。
**正确**：只有uc和il满足！其他量必须画0+等效电路求！

### 陷阱3：三要素公式符号错

**错误**：f(t)=f(∞)-[f(0+)-f(∞)]e^(-t/τ)（符号反了）
**正确**：f(t)=f(∞)+[f(0+)-f(∞)]e^(-t/τ)

### 陷阱4：非状态变量直接套三要素公式

**错误**：求vr(t)时直接套三要素公式。
**正确**：三要素公式只直接适用于状态变量（uc/il），非状态变量要先用三要素法求uc/il，再用VCR求。

## 练习建议

学完STEP法后，建议按以下顺序练习：

```
1. 基础练习（10题）：
   - 只用三要素法
   - 只求状态变量（uc/il）
   - 目的是熟练三要素计算

2. 进阶练习（10题）：
   - 用三要素法求非状态变量
   - 目的是掌握"先求uc/il，再求其他"的流程

3. 综合练习（5题）：
   - 混合零输入/零状态/全响应
   - 目的是理解三种响应的关系

4. 实战练习（3题）：
   - 复杂电路（含受控源、多个电阻）
   - 目的是熟练求Rth
```
