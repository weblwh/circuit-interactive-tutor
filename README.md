# 电路分析基础 交互式学习导师 (Circuit Analysis Interactive Tutor)

> 基于《电路基础》教材全13章，面向大学电子/电气工程专业学生的 AI 交互式学习系统。

## 功能特性

- **全教材覆盖**：13章完整课程体系，从电路基础知识到二端口网络
- **教材原图教学**：全部教材原始电路图和公式图（WMF已批量转换为PNG并压缩优化），禁止使用ASCII art
- **153道教材例题**：所有测验题必须来自教材例题，禁止自编题目
- **微步教学模型**：每个知识点拆分为2-6个微步骤，每步≤100字
- **掌握度追踪**：连续答对达到阈值才算掌握，自适应调整难度
- **学生注册系统**：支持学校+学号注册或匿名学习
- **学习历史记录**：记录每次学习的进度、正确率、薄弱点
- **智能学习建议**：根据进度给出下一步学习建议
- **学习数据上报**：每5次交互自动上报学习数据到云端

## 教学方法

参考 Khan Academy 的苏格拉底式教学法和 GCSE Physics Tutor 的分步计算法，调整为大学水平：

1. **微步教学**：每个知识点拆成超小步骤（每步不到100字）
2. **即时测验**：每步停下来确认理解，学完后用教材例题测验
3. **掌握度追踪**：连续答对2-3题才算掌握
4. **自适应调整**：根据表现决定继续还是复习

## 安装方法

### 方法一：在 WorkBuddy 中直接安装（推荐）

将本仓库的 GitHub 链接发给 WorkBuddy，对它说：

> 请帮我安装这个 skill：https://github.com/weblwh/circuit-interactive-tutor

WorkBuddy 会自动克隆并安装到 `~/.workbuddy/skills/circuit-interactive-tutor/`。

### 方法二：手动安装

```bash
# 1. 克隆仓库
git clone https://github.com/weblwh/circuit-interactive-tutor.git

# 2. 复制到 WorkBuddy skills 目录
cp -r circuit-interactive-tutor ~/.workbuddy/skills/

# 3. 重启 WorkBuddy 即可使用
```

## 使用方法

1. 在 WorkBuddy 对话中输入 `/circuit-interactive-tutor` 启动
2. 首次使用需注册（输入学校+学号或选择匿名）
3. 每天选择要学习的章节和节
4. 跟随微步教学流程学习
